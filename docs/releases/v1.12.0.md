# toolu v1.12.0

Released: 2026-06-14

## Highlights

This release trims the context footprint toolu injects into every session and fixes comemory scope inside git worktrees.

- **Leaner injected context.** The harness now trims and guards its own injected context footprint via four config-driven levers, cutting the per-session token cost toolu adds while preserving behaviour. (`toolu` 1.11.0 → 1.12.0)
- **Worktree-correct comemory.** `code-intel` memory is scoped by repo via the git common dir rather than the worktree path, so saved memories no longer vanish when working from a git worktree.

## Upgrade notes

No action needed for the context-footprint trim — it ships with the `toolu` core. To pick up the comemory worktree fix, update `code-intel`:

```text
/plugin install code-intel@falconiere
```
